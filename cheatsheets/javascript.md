# Add class

```javascript
elm.classList.add("className");
```

```javascript
elm.classList.add("first-class", "second-class", "third-class");
```

```javascript
elm.classList.remove("className");
```

```javascript
elm.classList.remove("first-class", "second-class", "third-class");
```
